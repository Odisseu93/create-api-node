# Node + Express + TypeScript API

Generated by [CNA - Create Node API](https://github.com/your-org/create-node-api).

## Setup

```bash
npm install
```

## Scripts

- `npm run dev` — Start dev server (ts-node)
- `npm run build` — Compile TypeScript to `dist/`
- `npm start` — Run compiled app (after build)
- `npm run lint` / `npm run lint:fix` — ESLint
- `npm run format` — Prettier
- `npm test` / `npm run test:watch` — Jest

## Environment

Copy `.env.example` to `.env` and set `PORT` and `NODE_ENV` as needed.
